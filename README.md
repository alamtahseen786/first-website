# 💜 Vanished — Let It Go

Vanished is a calming emotional wellness website created using HTML and CSS.  
The purpose of this project is to give users a peaceful space where they can anonymously express their thoughts, emotions, worries, and feelings without fear of being judged.

Instead of creating a basic beginner project, this website focuses on emotional comfort, aesthetic design, and user experience.

---

# Project Idea

Sometimes people carry emotions they never say out loud.

This project was designed around the idea:

> “Write it. Release it. Leave lighter.”

Users can:
- reflect on their emotions
- write what they feel
- mentally “release” their worries

>Nothing is saved, making the experience feel safe and temporary.

## 🖼️ Quote Images
Motivational and aesthetic quote images are displayed to emotionally comfort users.

## 💬 Comfort Quotes
